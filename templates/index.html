Welcome to the Cricket Match Data

About the Project

This project is part of the DAB111 course, focusing on the collection, processing, and presentation of cricket match data using Python, SQLite, and Flask.

It presents data about various Asia Cup cricket matches, analyzing the performance of teams, players, and their statistics.

About Dataset: Asia Cup CSV

The dataset used in this project contains information about the Asia Cup cricket tournaments. Below are the key details about the data:

  • Team: One team of the match
  • Opponent: Other team of the match
  • Format: Format of the tournament (ODI/T20I)
  • Ground: Venue of the match
  • Year: Tournament Year
  • Toss: Win/Lose the toss
  • Selection: Selection After winning/losing toss
  • Run Scored: Total runs scored
  • Wicket Lost: Total wickets lost
  • Fours: Total fours scored
  • Sixes: Total sixes scored
  • Extras: Extras given by opponent team
  • Run Rate: Batting innings run rate
  • Avg Bat Strike Rate: Average batting strike rate
  • Highest Score: Highest individual run scored
  • Wicket Taken: Opponent team's total wickets taken
  • Given Extras: Extras given to the opponent team
  • Highest Individual Wicket: Highest individual wickets taken by the opponent team
  • Player Of The Match: Player of the match
  • Result: Winning/Losing the match
